# Matur Dokter UI/UX Redesign

Repositori ini berisi implementasi **UI/UX redesign** dari aplikasi *Matur Dokter* dengan fokus pada tampilan antarmuka yang lebih modern, konsisten, dan mudah dikembangkan secara teknis.

Proyek ini ditujukan sebagai eksplorasi front-end dan prototype antarmuka, bukan sebagai aplikasi backend yang sudah terhubung ke sistem produksi.

## Tujuan
- Menyusun ulang struktur UI agar lebih rapi dan scalable
- Menyediakan komponen antarmuka yang mudah dikembangkan
- Mengelola asset desain (ikon, gambar, ilustrasi) secara terstruktur

## Fitur
- Struktur front-end berbasis Vite
- Komponen UI modular
- Asset gambar dan ikon terorganisir
- Konfigurasi pengembangan dengan hot reload

## Teknologi
- Vite
- HTML, CSS, JavaScript / TypeScript
- Struktur front-end modular

## Menjalankan Project

Pastikan Node.js sudah terpasang di sistem.

Run `npm i` untuk install dependensi.
Run `npm run dev` untuk mulai development server.

```bash
npm install
npm run dev
```

Aplikasi dapat diakses melalui browser pada alamat lokal yang ditampilkan di terminal.

## Struktur Folder:
```
src/
  assets/        # gambar dan ikon
  components/    # komponen UI
package.json
vite.config.ts
```

## Catatan
Repositori ini berfokus pada implementasi tampilan dan struktur front-end.
Pengembangan lebih lanjut seperti integrasi backend, API, dan autentikasi dapat ditambahkan sesuai kebutuhan.
